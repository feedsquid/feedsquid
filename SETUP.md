## Development setup

1. Install [devenv](https://devenv.sh/) &
   [direnv](https://github.com/direnv/direnv)
2. Clone git repository
3. Open project in your favorite IDE and see devenv doing some magic
4. Start dev server at http://localhost:8000
   ```bash
   deno task start
   ```
